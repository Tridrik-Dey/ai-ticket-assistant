# AI Ticket Assistant

An AI-powered system that classifies support tickets, extracts key information, and generates suggested responses.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Drizzle ORM + PostgreSQL
- OpenAI API (planned)

## Features (MVP)
- Ticket classification
- Priority detection
- AI-generated responses
- Smart routing

🚧 In development (Hackathon Project)
